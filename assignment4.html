Challenge 5: Accessibility Statement

One accessibility issue of my chessboard lies in the fact that it may be too small to see the pieces. Those with visual impairments may have trouble seeing the pieces and differentiating them.

Given this, I would most likely use different images rather than unicode characters for the pieces, or I would make the board + symbols larger.

Moreover, there may be an accessibility issue with the color picker user input. If someone were to use a screenreader, it would be near impossible to properly fill out all of the form.

I would remedy this with JavaScript, since it would allow for a user to better navigate the module.
